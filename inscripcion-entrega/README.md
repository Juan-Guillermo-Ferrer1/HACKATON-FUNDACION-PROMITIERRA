## 🚀 ¿Cómo inscribirse?

1. Ve a la pestaña **Issues** en GitHub
2. Click en **New issue**
3. Selecciona la plantilla **"Inscripción Participante"**
4. Completa todos los campos
5. Envía el formulario

Tu inscripción aparecerá automáticamente en la lista de [Participantes](../Participantes/README.md).
